# verdaccio-library

verdaccio-library is private local npm.

## Installation

install verdaccio globally.

```bash
sudo npm i verdaccio -g
```

## Usage

How to run verdaccio.

```bash
verdaccio
```

How to add user.

```bash
yarn addUser
```

How to publish.

```bash
yarn publish
```

How to use it
```bash
yarn add --S --registry http://localhost:4873 verdaccio-library
```
